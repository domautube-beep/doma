# SNS 마케팅 콘텐츠 생성기

제품/서비스/콘텐츠를 홍보하는 SNS 캡션과 해시태그를 생성한다.

## 사용자 입력 처리

1. **플랫폼** 파악 (인스타/트위터/유튜브/틱톡)
2. **목적** 파악 (홍보/공지/일상/바이럴)
3. **톤** 파악 (전문적/캐주얼/위트/감성)
4. 지정 안 하면 인스타 기준, 캐주얼 톤으로 자동 설정

## 플랫폼별 출력 형식

### 인스타그램

```
[훅 라인 — 첫 줄에 시선 잡기]

[본문 2-4줄 — 핵심 메시지]

[CTA — 행동 유도]

---
해시태그 (최대 15개):
#태그1 #태그2 #태그3 ...
```

규칙:
- 첫 줄이 피드에서 보이는 유일한 줄 → 강력한 훅 필수
- 이모지 적절히 사용 (과하지 않게)
- CTA 예: "링크는 프로필에서", "저장해두세요", "친구 태그해봐"

### 트위터/X

```
[본문 — 280자 이내]

[해시태그 2-3개]
```

규칙:
- 짧고 강렬하게
- 스레드용이면 1/N 형식으로 분할

### 유튜브

```
제목: [50자 이내, 클릭 유도]
설명:
[첫 2줄 — 핵심 요약]
[타임스탬프 (있으면)]
[링크/CTA]

태그: tag1, tag2, tag3 (최대 10개)
```

### 틱톡

```
[캡션 — 150자 이내, 후킹]
#해시태그1 #해시태그2 #해시태그3
```

## 톤 가이드

| 톤 | 특징 | 예시 |
|---|---|---|
| 전문적 | 신뢰감, 데이터, 권위 | "AI 음악 시장이 2026년 $4.2B를 돌파했습니다" |
| 캐주얼 | 친근, 대화체 | "이거 진짜 미쳤는데... AI가 만든 노래 들어봄?" |
| 위트 | 유머, 반전 | "작곡가 경력 0일차인데 첫 곡이 1만 스트리밍 찍음" |
| 감성 | 스토리텔링, 공감 | "새벽 3시, 이어폰 하나로 만든 첫 번째 노래" |

## AI 음악/크리에이터 특화 해시태그

```
#음악추천 #신곡 #자작곡 #AI크리에이터 #음악스타그램
#뮤직프로듀서 #비트메이킹 #가사쓰기 #songwriting
```

## 핵심 원칙

1. **첫 줄이 전부.** 피드에서 보이는 건 첫 줄뿐 — 스크롤을 멈추게 하라
2. **CTA 필수.** "좋아요/저장/공유/댓글" 중 하나는 유도
3. **해시태그는 정확하게.** 인기태그 + 니치태그 섞기
4. **3개 버전 제공.** 톤이 다른 3개 버전을 줘서 선택하게 한다
