# 디자인 프로젝트 견적서/청구서 생성기

BX/UX/UI/사이니지/웹/인쇄물 디자인 프로젝트의 견적서와 청구서를 생성한다.

## 사용자 입력 처리

1. 프로젝트 종류 파악 (BX, 웹, 앱, 사이니지, 인쇄물 등)
2. 범위/항목 확인
3. 아래 형식으로 문서 생성

## 견적서 형식

```markdown
# 견적서

**프로젝트명:** [프로젝트명]
**의뢰처:** [클라이언트명]
**작성일:** [날짜]
**작성자:** [하이컨시 / 담당: 사용자]

---

## 1. 프로젝트 개요
[프로젝트 배경 및 목적 2-3줄]

## 2. 작업 범위 및 비용

| No. | 항목 | 상세 내용 | 수량 | 단가(원) | 소계(원) |
|-----|------|----------|------|---------|---------|
| 1 | [항목명] | [상세] | 1식 | 0,000,000 | 0,000,000 |
| 2 | [항목명] | [상세] | 1식 | 0,000,000 | 0,000,000 |

**합계:** 0,000,000원
**부가세(10%):** 0,000,000원
**총액:** 0,000,000원

## 3. 일정

| 단계 | 기간 | 산출물 |
|------|------|--------|
| 기획/리서치 | 0주 | 기획서 |
| 디자인 | 0주 | 시안 |
| 수정/보완 | 0주 | 최종안 |
| 납품 | 0주 | 최종 파일 |

## 4. 정산 조건
- 선금 50%: 계약 체결 시
- 잔금 50%: 최종 납품 완료 후

## 5. 포함/불포함 사항

**포함:**
- [포함 항목]

**불포함:**
- 인쇄/제작비
- 촬영비
- 카피라이팅 (별도 협의)

## 6. 유효기간
본 견적서는 발행일로부터 30일간 유효합니다.
```

## 디자인 단가 참고 (2026년 시장 기준)

| 항목 | 단가 범위 |
|------|----------|
| BI/CI 개발 | 500만~3,000만 |
| BX 가이드라인 | 300만~1,500만 |
| 웹 메인 디자인 (PC+M) | 300만~800만 |
| 웹 서브페이지 | 50만~200만/페이지 |
| 앱 UI 디자인 | 500만~2,000만 |
| 사인 시스템 기획 | 1,000만~3,000만 |
| 사이니지 기획설계 | 500만~2,000만 |
| 인쇄물 (표지 시안) | 50만~200만/건 |
| 배너/SNS 디자인 | 10만~50만/건 |

이 단가는 참고용이며, 사용자가 직접 금액을 지정하면 그대로 따른다.

## 청구서 형식

견적서와 동일 구조에 아래 항목 추가:

```markdown
## 청구 정보
- **청구 금액:** 0,000,000원
- **청구 사유:** [선금/중도금/잔금] 정산
- **입금 계좌:** [은행명] [계좌번호] [예금주]
- **입금 기한:** [날짜]
```

## 출력 형식

사용자가 별도 요청 없으면 **마크다운**으로 출력한다.
"파일로 줘" 라고 하면 `.md` 파일로 저장한다.
"엑셀로" 라고 하면 CSV 형태로 정리한다.
